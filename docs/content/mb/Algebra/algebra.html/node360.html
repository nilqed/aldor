MonogenicLinearArithmeticType


Usage

MonogenicLinearArithmeticType R: Category


Parameter Type Description
R ExpressionType The coefficient domain
  ArithmeticType  


Description

MonogenicLinearArithmeticType is a common category for commutative and noncommutative univariate polynomials and power series with coefficients in an arbitrary arithmetic system R and with respect to an arbitrary basis . Its elements are not assumed to have finite support, so this type cannot be asserted to be an Algebra R even when R is a Ring.
