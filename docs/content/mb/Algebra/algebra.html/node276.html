MatrixCategory


Usage

MatrixCategory R: Category


Parameter Type Description
R ExpressionType The coefficient domain
  ArithmeticType  


Description

MatrixCategory R is a common category for matrices of arbitrary sizes with coefficients in R. They are -indexed and whether they do bound checking depends on each particular matrix type.
