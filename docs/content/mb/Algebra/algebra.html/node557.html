Shell


Usage

import from Shell(P, R, E)


Parameter Type Description
P Parser A parser type for the input
R PartialRing Resulting type of the evaluation
E Evaluator R An interpreter with target type R


Description

Shell(P, R, E) provides a basic read-eval-loop that converts its input to expressions of type R.


Exports

center: (TW, String) TW center a line of text
shell!: (P, TW, TW, SymbolTable R, Boolean, Boolean, String) Z Read-eval loop


where

TW == TextWriter
Z == MachineInteger
