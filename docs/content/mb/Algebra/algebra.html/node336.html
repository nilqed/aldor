fractionalRoots,roots


Usage

fractionalRoots(P)(p)
roots(P)(p)


Signature

fractionalRoots,roots: (P: POL %) P Generator FractionalRoot %


where

POL == UnivariatePolynomialCategory0


Parameter Type Description
P UnivariatePolynomialCategory0 % A polynomial type
p P A polynomial


Returns

Returns a generator that produces all the roots either in the ring or in its fraction field.
