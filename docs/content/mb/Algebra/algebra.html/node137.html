ChineseRemaindering


Usage

import from ChineseRemaindering R


Parameter Type Description
R EuclideanDomain an Euclidean Domain.


Description

ChineseRemaindering provides Garner's Chinese Remaindering Algorithm implemented over an arbitrary EuclideanDomain.


Exports

combine: (R,R) (R,R) R combine interpolated result with new modulus.
interpolate: (List R,List R) R interpolate given residues and moduli.


if R has IntegerCategory then

combine: (R,MachineInteger) (R,MachineInteger) R combine with new modulus.
