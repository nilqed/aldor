apply


Usage

A(n, m)
apply(A,n,m)
A(n, m, r, c)
apply(A,n,m,r,c)
A(a, b)
apply(A, a, b)


Signatures

apply: (%,MachineInteger,MachineInteger) R
apply: (%,MachineInteger,MachineInteger, MachineInteger,MachineInteger) %
apply: (%, Array MachineInteger, Array MachineInteger) %


Parameter Type Description
A % A matrix
n,m,r,c MachineInteger indices
a,b Array MachineInteger indices


Returns

A(n,m) returns the entry of A at its row and column, while A(n,m,r,c) returns the submatrix of A having in its top-left corner and rows and columns and returns the submatrix of A consisting of the intersection of the rows and columns of A.
