bound,finite?


Usage

bound s
finite? s


Signatures

bound: % MachineInteger
finite?: % Boolean


Parameter Type Description
s % a sequence


Returns

finite?(s) returns true if s is known to have finite support and false otherwise, while bound(s) returns if s is known to have finite support and for , otherwise.
