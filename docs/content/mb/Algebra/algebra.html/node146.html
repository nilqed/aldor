FractionalRoot


Usage

import from FractionalRoot


Description

FractionalRoot(R) provides fractions of R with multiplicities.


Parameter Type Description
R CommutativeRing A ring


Exports

ExpressionType
fractionalRoot: (R, R, Integer) % Create a root
integral?: % Boolean Test whether root is integral
integralRoot: (R, Integer) % Create a root
integralValue: % R Value of an integral root
multiplicity: % Integer Multiplicity of a root
setMultiplicity!: (%, Integer) % Change a multiplicity
value: % (R, R) Value of a root
