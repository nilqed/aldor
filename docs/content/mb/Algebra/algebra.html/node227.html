Backsolve


Usage

import from Backsolve(R,M)


Parameter Type Description
R IntegralDomain A coefficient ring
M MatrixCategory R A matrix type over R


Description

 (R,M) provides operations for backsolving triangular systems


Exports

backsolve: (M,Z Z,ARR Z,Z,Z,R) V R Backsolve a triangular system
backsolve: (M,Z Z,ARR Z,Z,M,R) (M, V R) Backsolve a triangular system


if R has GcdDomain then

backsolve: (M,Z Z,ARR Z,Z,Z) V R Backsolve a triangular system
backsolve: (M,Z Z,ARR Z,Z,M) (M, V R) Backsolve a triangular system
backsolve2: (M,Z Z,ARR Z,Z,Z,R) V R Backsolve a triangular system
backsolve2: (M,Z Z,ARR Z,Z,M,R) (M, V R) Backsolve a triangular system


where

Z == MachineInteger
ARR == PrimitiveArray
V == Vector
