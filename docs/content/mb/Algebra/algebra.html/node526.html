prefix


Usage

prefix(p, t, paren?, farg, op, left, right)
prefix(p, [], farg, op, left, right)


Signatures

prefix: (TEXT, TREE, TREE Boolean, (TEXT, TREE) TEXT,
     String, String, String) TEXT
prefix: (TEXT, List TREE, (TEXT, TREE) TEXT,
     String, String, String) TEXT


Parameter Type Description
p TEXT The port to write to
[] List TREE The arguments of the operator
paren? TREE Boolean The parenthetization function
farg (TEXT, TREE) TEXT The function for the arguments
op String The prefix symbol
left String The left parenthesis (optional)
right String The right parenthesis (optional)


where

TEXT == TextWriter
TREE == ExpressionTree


Description

Writes to , calling on each argument. Uses and , which default to ``('' and ``)'' for parenthetizing. The unary version calls to decide whether to parenthetize the argument.
