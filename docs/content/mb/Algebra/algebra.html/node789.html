entry,explode,key


Usage

entry p
(k, v) := explode p
key p


Signatures

entry: % V
explode: % (K, V)
key: % K


Parameter Type Description
p % a key-entry pair


Returns

entry(p) and key(p) return respectively the entry and key of p, while explode(p) returns the pair (key p, entry p).
