FractionByCategory


Usage

FractionByCategory R: Category


Parameter Type Description
R IntegralDomain an integral domain


Description

FractionByCategory(R) is the category of fractions of the integral domain R by some nonzero nonunit , i.e.the set of all fractions whose denominator is a power of p.
