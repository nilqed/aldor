char,ord


Usage

char n
ord c


Signatures

char: MachineInteger %
ord: % MachineInteger


Parameter Type Description
n MachineInteger a character code
c % a character


Returns

char(n) returns the character whose code is n, while ord(c) returns the code corresponding to the character c.
