LazyHalfWordSizePrimes


Usage

import from LazyHalfWordSizePrimes


Description

LazyHalfWordSizePrimes implements functionalities to obtain and manipulate half-word-size odd primes for lazy algorithms. Those primes are a few bits less that half-word-size, which allows for accumulation before reduction. Only a specific set of half-word-size primes is available.
