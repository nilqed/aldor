fft!


Usage

fft!(P)(r,p,q)


Signature

fft!: (P: UnivariatePolynomialCategory0 %) (P, P, P) Boolean


Parameter Type Description
P UnivariatePolynomialCategory0 % A polynomial type
r,p,q P Polynomials


Returns

Replaces by where the product is computed using FFT. The space occupied by the first argument is allowed to be reused. Returns true if the product could not be computed by the FFT method, false otherwise.
