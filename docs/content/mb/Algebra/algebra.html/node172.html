divisors


Usage

for d in divisors p repeat { ...}


Signature

divisors: % Generator R


Parameter Type Description
p % A product


Description

This generator yields all the products of the form where and .


Example

import from Integer, Product Integer, List Integer;

p := term(3, 1) * term(2, 2) * term(5, 2)       -- p = 3^1 2^2 5^2 = 300
l := sort! [divisors p];
creates the list 

  [1,2,3,4,5,6,10,12,15,20,25,30,50,60,75,100,150,300]
of all the divisors of .
