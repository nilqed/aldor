kernel!


Usage

kernel!(m, f)


Signature

kernel!: (M, M M) M


Parameter Type Description
m M A matrix
f M M Computes a kernel


Returns

Returns a basis for the kernel of m, uses a specialized algorithm if m is overdetermined, uses f when the system is no longer overdetermined.


Remarks

Can destroy m.
