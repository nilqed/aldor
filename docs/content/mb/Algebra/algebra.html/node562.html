set!


Usage

set!(t, x, v)
t.x := v


Signature

set!: (%, Symbol, T) T


Parameter Type Description
t % A symbol table
x Symbol A variable name
v T A value


Description

Assigns the value v to x in t. If x already had a value in t, the older value is lost.


Returns

Returns v.
