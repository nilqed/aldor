denominators


Usage

denominators(a,p,r,st)


Signature

denominators: (M,Z Z,Z, PrimitiveArray Z) PrimitiveArray R


where

Z == MachineInteger


Parameter Type Description
a M A matrix in REF form
p MachineInteger MachineInteger A permutation of the rows of a
r MachineInteger The number of stairs of the REF
st PrimitiveArray MachineInteger The stairs of the REF


Description

(a,p,r,st) must be the representation of a REF computed by rowEchelon, extendedRowEchelon or their bang-versions. When d is returned then for we have that times the -th column of the REF is a linear combination of the first leading columns of the REF. (The -th column is called leading if is a stair)
