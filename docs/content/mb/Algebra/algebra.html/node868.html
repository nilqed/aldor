VersionInformationType


Usage

VersionInformationType: Category


Description

VersionInformationType is the category of types providing version information.


Exports

major: MachineInteger major version number
minor: MachineInteger minor version number
name: String library name
patch: MachineInteger patch version number
version: String version information
