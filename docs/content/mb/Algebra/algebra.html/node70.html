ground?,term?


Usage

ground? m
term? m


Signature

ground?,term?: % Boolean


Parameter Type Description
m % An element of the module


Returns

ground?(m) returns true if is an element of R, false otherwise, while term?(m) returns true if for and an element of the basis, false otherwise.


Remarks

term?(0) returns true.
