pivot


Usage

pivot(a, p, c, r)


Signature

pivot: (M,Z Z, Z, Z) Z


where

Z == MachineInteger


Parameter Type Description
a M A matrix
p MachineInteger MachineInteger A permutation
c MachineInteger A column index
r MachineInteger A row index


Returns

Returns the row index of the an appropriate pivot for column at row or below. The matrix considered is with its rows permuted by .
