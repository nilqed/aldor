new


Usage

new n


Signature

new: MachineInteger %


Parameter Type Description
n MachineInteger a nonnegative size


Returns

Returns a string of null characters. This is useful when creating a memory buffer of a specified number of bytes, or when creating a string to be used as an TextWriter.
