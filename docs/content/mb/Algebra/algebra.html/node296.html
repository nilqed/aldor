zero


Usage

zero(n,m)
zero! A
zero? A


Signatures

zero: (MachineInteger, MachineInteger) %
zero!: % ()
zero?: % Boolean


Parameter Type Description
n,m MachineInteger integers
A % A matrix


Description

zero(n,m) returns an by zero matrix, while zero!(A) fills with 's and zero?(A) returns true if all the entries of A are , false otherwise.
