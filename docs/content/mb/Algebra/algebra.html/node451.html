FiniteVariableType


Usage

FiniteVariableType: Category


Description

FiniteVariableType is a category for multivariate polynomial variables that belong to a finite set. Hence, a domain of this category implements a finite set of ordered variables. The operations variable and index define a one-to-one map from % onto a range of (machine) integers.


Exports

VariableType
variable: MachineInteger % Retuns the -th variable of the type, if any
index: % MachineInteger Retuns the associated machine integer
#: MachineInteger The number of elements of the type
max: % The greatest element of the type
min: % The smallest element of the type
minToMax: List % The elements of the type sorted in increasing order
maxToMin: List % The elements of the type sorted in decreasing order
