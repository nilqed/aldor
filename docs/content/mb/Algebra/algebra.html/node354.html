random


Usage

random(n[, m])
random(n, f[, m])


Signatures

random: (Integer, Integer) %
random: (Integer, () R, Integer) %


Parameter Type Description
n Integer The desired degree
f () R A random generator for R
m Integer The desired number of terms (optional)


Returns

Returns a monic random polynomial of degree n with at most m terms, ( terms if m is not present). Uses to generate the coefficients if the parameter f is present, the random function otherwise.
