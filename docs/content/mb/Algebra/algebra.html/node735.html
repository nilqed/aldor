TextReader


Usage

import from TextReader


Description

TextReader provides various input streams.


Exports

push!: (C, %) () push back a character
read!: % C read one character
readUntil!: (StringBuffer, %, C) C read several characters
readWhile!: (StringBuffer, %, C Boolean) C read several characters
skipUntil!: (%, C) C skip characters
skipWhile!: (%, C Boolean) C skip characters
stdin: % standard input stream
textReader: (() C, C ()) % create a stream


where

C == Character
