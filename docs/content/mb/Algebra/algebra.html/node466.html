Description

PolynomialRing0 (R,V) is the category of the domains that implement a polynomial ring with coefficients in and variables in If is a finite set this polynomial ring is just If is finite or not, the set of monomials is the free abelian monoid generated by Moreover the default total ordering endowing is the the lexicographical one induced by Observe that the domain is not assumed to satisfy VariableType. In fact, only weaker conditions are required. Hence it is possible to use any totally ordered set as a domain of variables. For instance a set of algebraically independent numbers. Observe that PolynomialRing0 (R,V) provides essentially operations related to the structure of a free algebra. For more sophisticated operations (differentiation, evaluation, ...) see the category constructor PolynomialRing.
