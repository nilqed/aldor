insert


Usage

insert(t, a)
insert!(t, a)


Signature

insert: (T, %) %


Parameter Type Description
t T an element to add
a % a dynamic data structure


Description

Adds t to a and returns the new structure. insert creates a new structure while insert! modifies a itself.
