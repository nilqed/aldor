UnivariateGcdRing


Usage

UnivariateGcdRing: Category


Description

UnivariateGcdRing is the category of rings which export a gcd algorithm for univariate polynomials over themselves.


Exports

GcdDomain

gcdUP!: (P:UnivariatePolynomialCategory0) (P, P) P Gcd
