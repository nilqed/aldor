UnivariateTaylorSeriesCategory2Poly


Usage

import from UnivariateTaylorSeriesCategory2Poly(R, RXX, RX)

UnivariateTaylorSeriesCategory2Poly(R, RXX, RX) provides conversion tools between polynomials and series.
