Evaluator


Usage

Evaluator R: Category


Parameter Type Description
R PartialRing Resulting type of the evaluation


Description

Evaluator R is a category of interpreters, i.e. types which convert expression trees into elements of R whenever possible.


Exports

eval!: (TEXT, TREE, SYM R) Interpret an arbitrary tree
evalLeaf!: (LEAF, SYM R) Interpret a leaf
evalOp!: (TEXT, TREE, SYM R) Interpret
evalPrefix!: (String, Z, List R, SYM R) Interpret , is prefix


where

Z == MachineInteger
LEAF == ExpressionTreeLeaf
== Partial R
SYM == SymbolTable
TEXT == TextWriter
TREE == ExpressionTree
