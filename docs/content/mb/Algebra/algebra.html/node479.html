is?


Usage

is?(t, op)


Signature

is?: (%, ExpressionTreeOperator) Boolean


Parameter Type Description
t % An expression tree
op ExpressionTreeOperator An operator


Returns

is?(t, op) returns true if t is of the form op(args), false otherwise.
