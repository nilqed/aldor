degree,finite?


Usage

degree s
finite? s


Signatures

degree: % Partial Integer
finite?: % Boolean


Parameter Type Description
s % a series


Returns

finite?(s) returns true if s is known to have finite support and false otherwise, while degree(s) returns if s is known to have finite support and , failed otherwise.
