PartialFunction


Usage

import from PartialFunction(R, S)


Parameter Type Description
R Type Contains the domain of the function
S Type Contains the image of the function


Description

PartialFunction provides partial functions from R to S, i.e.functions from R to S which are allowed to fail on some elements of R.


Exports

apply: (%, R) S Apply a partial function
inDomain?: (%, R) Boolean Check if an element is in the domain
mapping: % (R S) Action of a function
partialApply: (%, R) Partial S Apply a partial function
partialMapping: % (R Partial S) Action of a function
predicate: % (R Boolean) Domain of a partial function
partialFunction: (R Partial S) % Create a partial function
partialFunction: ((R Boolean, R S)) % Create a partial function
