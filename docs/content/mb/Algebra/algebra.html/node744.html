textWriter


Usage

textWriter f
textWriter(f, g)


Signatures

textWriter: (Character ()) %
textWriter: (Character (), () ()) %


Parameter Type Description
f Character () the single-character write function
g () () the flush function (optional)


Returns

Returns the output stream for which writes the character and such that flushes the stream. If g is not given, then flushing the resulting stream has no effect.
