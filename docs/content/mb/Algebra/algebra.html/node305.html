firstDependence!


Usage

firstDependence!(gen, n, p, a, d)


Signature

firstDependence!: (Generator A Z, Z, Z, A A Z, A A Z) Z


where

A == PrimitiveArray
Z == MachineInteger


Parameter Type Description
gen Generator A MachineInteger A generator of vectors
n MachineInteger The dimension of the vectors generated
p MachineInteger a prime
a A A MachineInteger A work matrix
d A A MachineInteger A matrix for the dependence


where

A == PrimitiveArray


Description

firstDependence!(gen,n,p,a,d) computes the first dependence over among the vectors generated by gen. It returns the number of vectors generated. This number is as small as possible, meaning that the first vectors are linearly independent over . The coefficients of the dependence are stored in the first column of , which must have at least rows, upon return. The work matrix must have rows and at least columns (note that ). The dimension of the vectors generated by gen must be n. There must be a relation between the vectors generated.
