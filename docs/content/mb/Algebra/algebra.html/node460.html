MachineIntegerExponentVectorCategory


Usage

MachineIntegerExponentVectorCategory V: Category


Parameter Type Description
V FiniteVariableType The type of variables


Description

MachineIntegerExponentVectorCategory V is a category for the exponents of the monomials (or power products) generated by the finite set of variables and coded as direct products of non-negative machine integers.


Exports

CopyableType
ExponentCategory V
HashType
free!: % () Asserts that the input will no longer be used
exponent: Tuple I % Creation from a tuple
exponent: PrimitiveArray I % Creation from a primitive array
    Degrees start at slot 1
    Slot must be the total degree
parray: % PrimitiveArray I Inverse mapping of exponent


where

I == MachineInteger
