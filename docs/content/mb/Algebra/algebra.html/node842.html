map!


Usage

map! f
map!(f)(s)


Signature

map!: (T T) % %


Parameter Type Description
f T T a map
s % a stream


Returns

map!(f)(s) returns the stream [f(x) for x in s], while map!(f) returns the mapping [f(x) for x in s]. In both cases, the stream s is modified in place and no copy is made.
