coerce,expand,trailExpand


Usage

p::%
coerce p
expand q
(n, h) := trailExpand q


Signatures

coerce: Rx %
expand: % Rx
trailExpand: % (Integer, Rx)


Parameter Type Description
p Rx A polynomial
q % A factorial polynomial


Description

p::% converts p from the power basis to the factorial basis , while expand(q) performs the reverse conversion and trailExpand(q) returns such that .
