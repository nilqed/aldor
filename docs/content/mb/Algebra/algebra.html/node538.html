MakePartialRing


Usage

import from MakePartialRing(R, f)


Parameter Type Description
R Ring A ring
f R Partial Integer A retraction to the integers


Description

MakePartialRing(R, f) is a partial ring isomorphic to R. The function is used to retract elements of this partial ring to integers whenever possible. This type can be used in order to build an evaluator that interprets expression trees into R.


Exports

PartialRing
coerce: R % Convert an element of R to one of the partial ring
coerce: % R Convert an element of the partial ring to one of R
