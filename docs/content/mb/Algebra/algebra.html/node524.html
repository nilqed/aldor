infix


Usage

infix(p, [], paren?, farg, op, left, right)


Signatures

infix: (TEXT, List TREE, TREE Boolean, (TEXT, TREE) TEXT,
     String, String, String) TEXT


Parameter Type Description
p TEXT The port to write to
[] List TREE The arguments of the operator
paren? TREE Boolean The parenthetization function
farg (TEXT, TREE) TEXT The function for the arguments
op String The infix symbol
left String The left parenthesis (optional)
right String The right parenthesis (optional)


where

TEXT == TextWriter
TREE == ExpressionTree


Description

Writes to , calling on each argument, and calling to decide whether to parenthetize each argument. Uses and , which default to ``('' and ``)'' when parenthetizing.
