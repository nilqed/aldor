Signatures

diagonal: Vector R %

Parameter Type Description
R Entries
n MachineInteger A size
A % A matrix


Description

diagonal([]) returns a diagonal matrix whose diagonal elements are , while diagonal() returns an diagonal matrix with r on its diagonal, and square?(A) (resp. diagonal A and scalar? A) return true if A is a square (resp. diagonal and diagonal with the same entry on its diagonal) matrix, false otherwise.
