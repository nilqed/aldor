one,one?


Usage

one n
one? A


Signatures

one: MachineInteger %
one?: % Boolean


Parameter Type Description
n MachineInteger an integer
A % A matrix


Description

one(n) returns an identity matrix, while one?(A) returns true if A is an identity matrix, false otherwise.
