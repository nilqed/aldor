texParen?


Usage

texParen?(prec, t)


Signature

texParen?: (MachineInteger, %) Boolean


Parameter Type Description
prec MachineInteger An operator precendence.
t % An expression tree


Returns

Returns true if should be parenthetized when appearing as argument of an operator of precedence prec, false otherwise.
