Data structures

All the data structure of Algebra are provided by libaldor. Some of them are extended by mathematical operations and take on a new name: Sequence corresponds to Stream from libaldor and Vector corresponds to Array from libaldor, in both cases with pointwise arithmetic operations added. Note however that Array is -indexed while Vector is -indexed.
