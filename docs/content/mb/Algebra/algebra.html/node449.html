ExponentCategory


Usage

ExponentCategory V: Category


Parameter Type Description
V VariableType The domain of variables


Description

ExponentCategory V provides multivariate monomials (i.e. products of variables from V) looked as an addtive ordered monoid (with cancellation) by associating to every product of variables its sequence of degrees.


Exports

GeneralExponentCategory
exponent: V % The exponent of a variable
exponent: (V, Z) % The exponent of a power of a variable
exponent: Generator Cross (V, Z) % The exponent of a power product
exponent: (List V, List Z) % The exponent of a power product
terms: % Generator Cross (V, Z) Inverse map of exponent
mainVariable: % Partial V The biggest variable, if any
variables: % List V The list of variables of a monomial
degree: (%, V) Z The degree w.r.t. a variable
totalDegree: % Z The sum of the degrees of an exponent
totalDegree: (%, List V) Z The sum of the degrees w.r.t. a list
gcd: (%, %) % Monomial gcd
lcm: (%, %) % Monomial lcm
syzygy: (%, %) % Cofactors w.r.t. lcm


where

Z == Integer
