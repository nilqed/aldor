Usage

Signature

Parameter Type Description

Remarks

order(a) makes some precalculations based on a, so if you need to use order(a)(b) several times with the same a and different b's, it is more efficient to compute order(a) once and assign it, as in the example below. Example

The following function computes the orders at of a list of polynomials :

}
