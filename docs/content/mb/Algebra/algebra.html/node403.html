UnivariatePolynomialCategory


Usage

UnivariatePolynomialCategory R: Category


Parameter Type Description
R ExpressionType The coefficient domain
  ArithmeticType  


Description

UnivariatePolynomialCategory is the category of univariate polynomials with coefficients in an arbitrary domain R and with respect to the power basis .
