User's Guide

This guide introduces the common types and categories provided by SALLI, and presupposes some familiarity with programming in ALDOR. If you are unfamiliar with ALDOR, then after installing SALLI, go through the tutorial salli/tutorial/tutorial.ps, which will familiarize you both with ALDOR and SALLI. SALLI provides basic categories and types in 3 areas: arithmetic, data structures and input/output. Figure 1 shows the category hierarchy in those 3 areas and a few of the types provided. Note that all the category names end with the word Type.
