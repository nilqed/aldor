generator




Signature

  • generator: % Generator T





Example

  • The following code computes the sum of all the elements of an array of machine integers: 
    sum(a:Array MachineInteger, n:MachineInteger):MachineInteger == {
    s:MachineInteger := 0;
    for x in a repeat s := s + x;
    s;
}
