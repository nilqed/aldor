generator




Signature

  • generator: % Generator Z





Example

  • The following code computes the sum of all the positive even machine integers that are smaller than : 
    evenSum(n:MachineInteger):MachineInteger == {
    s := 0;
    for x in 2..prev(n) by 2 repeat s := s + x;
    s;
}
