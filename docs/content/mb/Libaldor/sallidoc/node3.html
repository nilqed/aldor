What is SALLI?

SALLI (the Standard ALdor LIbrary) is a compact replacement for the axllib library that is provided by NAG together with the ALDOR compiler. It provides a low-level interface between ALDOR programmers and the abstract machine, allowing its users to start programming with a minimal set of basic types and data structures already built into the language.
