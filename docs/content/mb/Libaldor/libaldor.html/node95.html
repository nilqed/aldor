RandomNumberGenerator


Usage

import from RandomNumberGenerator


Description

RandomNumberGenerator provides independent pseudo-random number generators.


Exports

apply: % Z generate a random number
generator: % Generator Z generate random numbers
max: % Z largest number that can be generated
min: % Z smallest number that can be generated
numberOfGenerators: Z number of predefined generators
randomGenerator: () % get a generator
randomGenerator: Z % get a generator
randomGenerator: (Z, Z) % get a bounded generator
randomGenerator: (Z, Z, Z) % get a bounded generator
randomInteger: () Z generate a random number
seed: (%, Z) Z set the seed


where

Z == MachineInteger
