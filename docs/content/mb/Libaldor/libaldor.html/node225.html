setRest!


Usage

setRest!(l,t)


Signature

setRest!: (%,%) %


Parameter Type Description
l % a nonempty list
t % a list


Description

Replaces l by the list [first(l),t] and returns t.


Remarks

setRest!(l,t) does not make a copy of l, which is therefore modified after the call.
