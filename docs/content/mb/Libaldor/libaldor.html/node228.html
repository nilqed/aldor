MemoryBlock


Usage

import from MemoryBlock


Description

MemoryBlock provides packed arrays of bytes, -indexed and without bound checking (the debug version of libaldor provides bound-checking for memory blocks).


Exports

ArrayType(Byte, PrimitiveMemoryBlock)
