apply


Usage

apply(t, k)
t.k


Signature

apply: (%, K) V


Parameter Type Description
t % a table
k K a key


Returns

Returns the element of t with key k, which must be present in the table.


Remarks

Produces an error if k is not in t, use find if it is not known whether k is present in table.
