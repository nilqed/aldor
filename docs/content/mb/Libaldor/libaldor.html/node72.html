gcd,lcm


Usage

gcd(a, b)
lcm(a, b)


Signature

gcd,lcm: (%,%) %


Parameter Type Description
a,b % integers


Returns

gcd(a, b) and lcm(a, b) return respectively a greatest common divisor and a least common multiple of a and b.
