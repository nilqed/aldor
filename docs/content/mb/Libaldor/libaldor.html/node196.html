HashTable


Usage

import from HashTable(K, V)
import from HashTable(K, V, h)


Parameter Type Description
K PrimitiveType the type of the keys
  HashType  
V Type the type of the entries
h K MachineInteger the hash function to use


Description

HashTable provides hash tables with keys of type , entries of type and that uses the hash-function . If has HashType, then the parameter is optional as the function hash is used by default in that case.


Exports

TableType(K, V)
forget: (K V) % creation of a table with default
remember: (K V) % creation of a table with default
remove!: (K, %) % removal of a key
