append!


Usage

append!(l, x)
append!(l, s)


Signatures

append!: (%, T) %
append!: (%, %) %


Parameter Type Description
l,s % lists
x T an entry


Returns

append!(l,x) and append!(l,s) return the lists and respectively.


Remarks

append! does not make a copy of , which is therefore modified after the call. It is unsafe to use the variable after the call, unless it has been assigned to the result of the call, as in l := append!(l, x).
