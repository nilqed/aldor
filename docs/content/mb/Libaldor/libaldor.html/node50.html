GMPFloat


Usage

import from GMPFloat


Description

GMPFloat implements arbitrary precision floats with the float arithmetic provided by GMP.


Exports

CopyableType
FloatType
coerce: SingleFloat % conversion
  DoubleFloat %  
  GMPInteger %  
defaultPrecision: () Z default precision
limbs: % Generator Z iteration
machine: % DoubleFloat conversion to double precision
new: () % initialization (for calling GMP)
precision: % Z precision
setDefaultPrecision: Z Z adjust default precision
setPrecision!: (%, Z) % adjust precision
single: % SingleFloat conversion to single precision


where

Z == MachineInteger
