by,step


Usage

s by n
step s


Signatures

by: (%,Z) %
step: % Z


Parameter Type Description
s % a segment
n Z a step


Returns

s by n changes s to become the segment consisting of every integer in its range, i.e.the integers that are within the range or of s, while step(s) returns n such that s is the segment consisting of every integer in its range.


Remarks

The function s by n does not create a new segment but side-effects s, whose former step is lost.
