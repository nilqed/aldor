precision,setPrecision!


Usage

precision x
setPrecision!(x, n)


Signatures

precision: % MachineInteger
setPrecision!: (%, MachineInteger) %


Parameter Type Description
x % a float
n MachineInteger a precision


Description

precision(x) returns the precision acutally used for , while setPrecision!(x, n) sets the precision for to be at least bits.


Remarks

Those functions are wrapper to the mpf_get_prec and mpf_set_prec GMP functions, so setPrecision! can involve a call to realloc.


See Also

defaultPrecision,setDefaultPrecision
