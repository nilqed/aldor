sort!


Usage

sort! a
sort!(a, f)


Signature

sort!: (%,(T,T) Boolean) %


Parameter Type Description
a % a primitive array
f (T, T) Boolean a comparison function


Description

Sorts the array using the ordering . The comparison function is optional if has TotallyOrderedType, in which case the order function of is taken.
