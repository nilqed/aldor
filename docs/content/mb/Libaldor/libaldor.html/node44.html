coerce,single


Usage

x::%
single y


Signatures

coerce: SingleFloat %
single: % SingleFloat


Parameter Type Description
x SingleFloat a single float
y % a double float


Description

coerce(x) returns x converted to a double float, while single(y) truncates y to a single precision float.
