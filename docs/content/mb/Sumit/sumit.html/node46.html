ordinary?,singular?,regular?,irregular?


Usage

ordinary? N
singular? N
regular? N
irregular? N


Signature

ordinary?,singular?,regular?,irregular?: % Boolean


Parameter Type Description
N % a Newton polygon


Description

When Rxy is a LinearOrdinaryDifferentialOperatorCategory over Rx, then those functions characterize the roots of the center of N with respect to the differential operator. They test whether those roots are ordinary points are singularities, and whether singularities are regular or irregular.


Remarks

Those functions are only meaningful for differential Newton polygons.
