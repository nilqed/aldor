ExteriorPower


Usage

import from ExteriorPower(R, n, m)


Parameter Type Description
R IntegralDomain The coefficient domain
n MachineInteger The power to compute
m MachineInteger Order of the operator


Description

ExteriorPower(R, n, m) provides functions that computes exterior powers of operators of order having invertible leading coefficients.
