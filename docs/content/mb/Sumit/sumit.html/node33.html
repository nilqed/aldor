apply


Usage

apply(p, t)
p t
apply(port, p, x)
port(p, x)


Signatures

apply: (%, ExpressionTree) ExpressionTree
apply: (TextWriter, %, Symbol) TextWriter


Parameter Type Description
p % An element of the type
t ExpressionTree An expression tree
x Symbol A name for the variables
port TextWriter An output port


Returns

apply(p, t) returns p as an expression tree, using t as root variable name.
apply(port, p, x) sends p to port using x as root variable name, and returns the output port afterwards.
