Signatures

where

EQ == LinearHolonomicDifferentialEquation(R, RX, RXD)


Parameter Type Description
L EQ A linear ordinary differential operator
  RXD  
p RX Allowed poles


Returns

kernel(L) returns such that is a basis for all the rational solutions of . If a nonzero second argument p is present, then only the solutions having their poles among the roots of p (and at infinity) are computed.


Remarks

d is a ``universal denominator'' in the sense that any rational solution must be of the form for a polynomial p. However, d is allowed to have negative exponents, hence it can be a fraction itself rather than a polynomial. Use expandFraction from Product to expand it properly.
