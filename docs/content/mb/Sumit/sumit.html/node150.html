kernel


Usage

kernel L
kernel(L, n)


Signatures

kernel: RXY Vector FX
kernel: (RXY, Integer) Vector FX


Parameter Type Description
L RXY A linear ordinary differential or recurrence operator
n Integer A degree bound (optional)


Description

kernel(L) returns a basis for all the polynomial solutions of , while kernel(L,n) returns a basis for the polynomial solutions of of degree at most .


Remarks

kernel(L) is available only if R has RationalRootRing. Produces an error if is neither a differential or recurrence operator. The polynomials in the kernel are expressed in terms of the basis described in recurrence.
