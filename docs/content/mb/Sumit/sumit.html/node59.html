exteriorPowerSystem,symmetricPowerSystem


Usage

exteriorPowerSystem L
symmetricPowerSystem L


Signature

exteriorPowerSystem,symmetricPowerSystem: Rd DenseMatrix R


Parameter Type Description
L Rd A differential operator of order m


Returns

Return a matrix M such that the differential system is respectively the exterior and symmetric power system of L.
