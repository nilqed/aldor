UnivariateSkewPolynomial


Usage

import from UnivariateSkewPolynomial(R, Rx, , )


Parameter Type Description
R Ring The coefficient ring
Rx UnivariatePolynomialCategory R A polynomial type over R
Automorphism R The automorphism to use
R R The -derivation to use


Description

UnivariateSkewPolynomial(R, Rx, , ) implements skew univariate polynomials with coefficients in R. Rx is used for representing the skew-polynomials, so you can choose between sparse and dense representations.
