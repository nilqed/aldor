LinearOrdinaryDifferentialOperator


Usage

import from LinearOrdinaryDifferentialOperator S;
import from LinearOrdinaryDifferentialOperator(S, x);
import from LinearOrdinaryDifferentialOperator(R, D);
import from LinearOrdinaryDifferentialOperator(R, D, x);


Parameter Type Description
R Ring The coefficient ring of the operators
S DifferentialRing The coefficient ring of the operators
D Derivation R The derivation to use for the action
x Symbol The variable name (optional)


Description

LinearOrdinaryDifferentialOperator(R, D, x) implements linear ordinary differential operators with coefficients in R. If the derivation D is not specified, then R must be a DifferentialRing. It is possible to override the default derivation on R simply by passing another one as parameter.
