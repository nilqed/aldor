UnivariateSkewPolynomialDecompositionCategory


Usage

UnivariateSkewPolynomialDecompositionCategory(R, F, , RX, RXZ, FX, FxZ, FxY):
Category


Parameter Type Description
R FactorizationRing The coefficient ring
F Field A field containing R
  FactorizationRing  
Injection from R into F
RX UnivariatePolynomialCategory R Polynomials over R
RXZ UnivariateSkewPolynomialCategory RX Operators over RX
FX UnivariatePolynomialCategory F Polynomials over F
FxZ UnivariateSkewPolynomialCategory Fraction FX Operators over Fx
FxY UnivariatePolynomialCategory Fraction FX For returning algebraics


Description

UnivariateSkewPolynomialDecompositionCategory(R, F, , RX, RXZ, FX, FxZ, FxY) is the category of types providing various decompositions of operators in RXZ.


Exports

decompose: RXZ Union(rat:RAT, alg:ALG) LCLM decomposition
factorDecomp: RXZ Union(rat:RAT, alg:ALG) Factorization
Loewy: RXZ Vector FxZ Loewy decomposition


where

I == MachineInteger
RAT == Record(mode:I, rat: Vector FxZ)
ALG == Record(mode:I, modulus:FX, c0:Fraction FxY, ratpart: Vector FxZ)
