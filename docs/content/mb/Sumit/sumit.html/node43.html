finite?, infinite?


Usage

finite? N
infinite? N


Signature

finite?, infinite?: % Boolean


Parameter Type Description
N % a Newton polygon


Returns

finite?(N) returns true if N is over a finite center, false if N is above infinity, while infinite?(N) returns the opposite.
