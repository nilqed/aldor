DenseHomogeneousExteriorPower


Usage

import from DenseHomogeneousExteriorPower(R, n, m)


Parameter Type Description
R Ring The coefficient ring of the exterior power
n MachineInteger The total degree
m MachineInteger The number of variables


Description

DenseHomogeneousExteriorPower (R, n, m) implements densely the exterior power of the free module of rank over R.


Exports

FiniteFreeModule(R, DenseHomogeneousExteriorProduct(n,m))
