Usage

import from LinearOrdinaryFirstOrderSystemPolynomialSolutions(R, F, , RX, RXY, FX)


Parameter Type Description
R IntegralDomain An integral domain
F Field A field containing R
Injection from R into F

FX MonogenicAlgebra F Polynomials over F
  CommutativeRing  


Description

LinearOrdinaryFirstOrderSystemPolynomialSolutions(R, F, , RX, FX) provides a polynomial solver for first order linear ordinary differential or recurrence systems with polynomial coefficients.
