LinearOrdinaryDifferentialSystemSeriesSolutions


Usage

import from LinearOrdinaryDifferentialSystemSeriesSolutions(R, F, , RX, Fx)


Parameter Type Description
R IntegralDomain An integral domain
F Field A field containing R
Injection from R into F
RX UnivariatePolynomialCategory R Polynomials over R
Fx UnivariateTaylorSeriesCategory F A series type over F


Description

LinearOrdinaryDifferentialSystemSeriesSolutions(R, F, , RX, Fx) provides a series solver for linear ordinary differential systems with polynomial coefficients around an ordinary point.


Exports

kernel: (SYS RX, R) V V Fx all the series solutions
solution: (SYS RX, R) V F V Fx V Fx solve an initial value problem


where

SYS == LinearOrdinaryFirstOrderSystem
V == Vector
