PowerSystem


Usage

exteriorPowerSystem(L, n)
symmetricPowerSystem(L, n)


Signatures

{exterior,symmetric}PowerSystem: (%, Z) LinearOrdinaryFirstOrderSystem R


where

Z == Integer


Parameter Type Description
L % A differential operator
n Integer A positive integer


Description

Return the exterior or symmetric power of the companion system associated with L.


See Also

companionSystem, exteriorPower,symmetricPower
