What is ?

is a computer algebra library designed to provide reusable and efficient implementations of algorithms for manipulating and solving linear ordinary differential and difference equations and systems. Built as an extension of the Algebra library, it provides ALDOR programmers with a extensible computer algebra layer with many low-level functions for handling skew-polynomials and their related skew-linear systems.
