LinearOrdinaryDifferentialOperatorExponentialSolutions


Usage

import from LinearOrdinaryDifferentialOperatorExponentialSolutions(R, F, , RX, RXE, FX)


Parameter Type Description
R FactorizationRing The coefficient ring
F Field A field containing R
  FactorizationRing  
Injection
RX UnivariatePolynomialCategory R Polynomials over R
RXD LinearOrdinaryDifferentialOperatorCategory RX Differential operators
FX UnivariatePolynomialCategory F Polynomials over F


Description

LinearOrdinaryDifferentialOperatorExponentialSolutions(R, F, , RX, RXD, FX) provides an exponential solver for linear ordinary differential operators with polynomial coefficients.


Exports

exponentialSolutions: EQ EXPSOLS Exponential solutions
  RXD EXPSOLS  


where

EQ == LinearHolonomicDifferentialEquation(R, RX, RXD)
EXPSOLS == List Cross(List Fraction FX, Product FX, Vector FX)
