NewtonPolygon


Usage

import from NewtonPolygon(R, Rx, Rxy)


Parameter Type Description
R IntegralDomain the ground ring
Rx UnivariatePolynomialCategory R polynomials over R
Rxy UnivariatePolynomialAlgebra Rx polynomials over Rx


Description

NewtonPolygon(R, Rx, Rxy) implements the algebraic Newton polygon for bivariate polynomials over R. When Rxy is a LinearOrdinaryDifferentialOperatorCategory over Rx, the points of the algebraic Newton polygon are in bijective correspondence with the points of the differential Newton polygon, so NewtonPolygon(R, Rx, Rxy) also provides functions related to the differential Newton polygon.
