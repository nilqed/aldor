LinearOrdinaryDifferentialSystemPolynomialSolutions


Usage

import from LinearOrdinaryDifferentialSystemPolynomialSolutions(R, F, , RX, FX)


Parameter Type Description
R GcdDomain The coefficient ring
  RationalRootRing  
F Field A field containing R
Injection from R into F
RX UnivariatePolynomialCategory R Polynomials over R
FX UnivariatePolynomialCategory F Polynomials over F


Description

LinearOrdinaryDifferentialSystemPolynomialSolutions(R, F, , RX, FX) provides a polynomial solver for first order systems of linear ordinary differential operators with polynomial coefficients.


Exports

kernel: SYS RX DenseMatrix FX Polynomial solutions


where

SYS == LinearOrdinaryFirstOrderSystem
