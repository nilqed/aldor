optimize


Usage

optimize(a,f)


Signature

optimize: (M, R Partial Z) (PrimitiveArray Z, PrimitiveArray Z, Z)


where

Z == MachineInteger


Parameter Type Description
a M A matrix
f R Partial MachineInteger A partial function


Description

This provides the optimization described in:
M. Bronstein, T. Mulders & J.A. Weil, On symmetric powers of differential operators, Proceedings of ISSAC'97, ACM Press (1997), 156-163.
