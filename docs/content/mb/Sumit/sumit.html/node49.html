NewtonPolygonEquations


Usage

import from NewtonPolygonEquations(R, Rx, Rxy, E, Ex)


Parameter Type Description
R IntegralDomain The coefficient ring
Rx UnivariatePolynomialCategory R Polynomials over R
Rxy UnivariatePolynomialAlgebra Rx polynomials over Rx
E CommutativeRing An extension of R
Ex UnivariatePolynomialCategory E Polynomials over E


Description

NewtonPolygonEquations(R, Rx, Rxy, E, Ex) provides tools to compute exponents and indicial and characteristic equations associated with Newton polygons.


Exports

exponentialPart: (Rxy, E, Integer, Integer, E, R E) Ex Exponential part
series: (Rxy, E, Integer, Integer, E, R E) Laurent series


where

== DenseUnivariateTaylorSeries E
