DenseHomogeneousPolynomial


Usage

import from DenseHomogeneousPolynomial(R, n, m)


Parameter Type Description
R Ring The coefficient ring of the polynomials
n MachineInteger The total degree
m MachineInteger The number of variables


Description

DenseHomogeneousPolynomial (R, n, m) implements dense homogeneous polynomials of total degree in variables with coefficients in R.


Exports

FiniteFreeModule(R, DenseHomogeneousPowerProduct(n,m))
