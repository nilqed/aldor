LinearOrdinaryRecurrenceHypergeometricSolutions


Usage

import from LinearOrdinaryRecurrenceHypergeometricSolutions(R, F, , RX, RXE, FX)


Parameter Type Description
R FactorizationRing A gcd domain
F Field A field containing R
Injection from R into F
RX UnivariatePolynomialCategory R Polynomials over R
RXE LinearOrdinaryRecurrenceCategory RX Recurrences over RX
FX UnivariatePolynomialCategory F Polynomials over F


Description

LinearOrdinaryRecurrenceHypergeometricSolutions(R, F, , RX, RXE, FX) provides a hypergeometric solver for linear ordinary recurrence operators with polynomial coefficients.

allHypergeometricSolutions: RXE List Cross(Fx, Vector FX) All solutions
hypergeometricSolution: RXE Partial Fx A solution
