UnivariateSkewPolynomialCategoryTools


Usage

import from UnivariateSkewPolynomialCategoryTools(R, C)


Parameter Type Description
R Ring The coefficient ring
C UnivariateSkewPolynomialCategory R A skew-polynomial type


Description

UnivariateSkewPolynomialCategoryTools(R, C) provides the univariate skew polynomial arithmetic operations which depend on the and functions determining the skew polynomial ring.


Exports

apply: (C, R, R, MOR, R R) R Apply a skew-polynomial to a scalar
times: (C, C, MOR, R R) C Product of skew-polynomials


if R has IntegralDomain then

leftExactQuotient: (C, C, MOR, R R) Partial C Left exact quotient
monicLeftDivide: (C, C, MOR, R R) (C, C) Left Euclidean division
monicRightDivide: (C, C, MOR, R R) (C, C) Right Euclidean division
rightExactQuotient: (C, C, MOR, R R) Partial C Right exact quotient


if R has Field then

leftDivide: (C, C, MOR, R R) (C, C) Left Euclidean division
rightDivide: (C, C, MOR, R R) (C, C) Right Euclidean division


where

MOR == Automorphism R
