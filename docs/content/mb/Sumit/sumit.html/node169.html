LinearOrdinaryRecurrenceSystemTools


Usage

import from LinearOrdinaryRecurrenceSystemTools(R, F, , RX)


Parameter Type Description
R IntegralDomain An integral domain
F Field A field containing R
An injection from R into F
RX UnivariatePolynomialCategory R A polynomial ring over R


Description

LinearOrdinaryRecurrenceSystemTools(R, F, , RX) provides tools for viewing linear recurrences with matrices of polynomials as coefficients, as generators of sequences of vectors satisfying the corresponding recurrences.
