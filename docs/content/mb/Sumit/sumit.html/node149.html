LinearOrdinaryOperatorPolynomialSolutions


Usage

import from LinearOrdinaryOperatorPolynomialSolutions(R, F, , RX, RXY, FX)


Parameter Type Description
R IntegralDomain An integral domain
F Field A field containing R
Injection from R into F
RX UnivariatePolynomialCategory R Polynomials over R
RXY UnivariateSkewPolynomialCategory RX Operators over RX
FX MonogenicAlgebra F Polynomials over F
  IntegralDomain  


Description

LinearOrdinaryOperatorPolynomialSolutions(R, F, , RX, RXY, FX) provides a polynomial solver for linear ordinary differential or recurrence operators with polynomial coefficients.
