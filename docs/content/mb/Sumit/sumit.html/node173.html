Parameter Type Description
L % A linear recurrence
e Integer An optional shift (default is 0)
a PrimitiveArray MachineInteger Will contain the singularities of
v Vector Vector F Will contain linear constraints


Description

Returns a generator for all the values generated by starting will all possible initial values. As vectors are being produced by the resulting generator, is being updated to contain the number of singularities encountered, while contain the indices of the encountered singularities, and contain the corresponding constraints on the initial values for the sequence to be a solution of . The kernel of is a subspace of the span of the sequence of vectors generated modulo the obtained constraints. It is exactly that space after all the singularities of have been encountered. It is possible to give empty as parameter for a (resp. v), in which case the singularities (resp. constraints) are not computed. Note that the size of the vectors generated increase by one at each singularity.
