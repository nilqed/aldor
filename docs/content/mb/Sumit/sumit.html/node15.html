DenseFreeModule


Usage

import from DenseFreeModule(R, V)
import from DenseFreeModule(R, V, x)


Parameter Type Description
R Ring The coefficient ring
V FiniteSet A basis of the module
x Symbol The variable name (optional)


Description

DenseFreeModule (R, V) implements densely the free module generated by V over R.
