particularSolution


Usage

particularSolution(L, g)


Signature

particularSolution: (RXD, Fraction RX) (Product RX, Partial FX)


Parameter Type Description
L RXD A linear ordinary differential operator
g Fraction RX A fraction


Returns

Returns (d, failed) if has no rational solution, in which case d is a universal denominator for such solutions. Otherwise, it returns such that is a particular solution.


Remarks

d is a ``universal denominator'' in the sense that any rational solution must be of the form for a polynomial p. However, d is allowed to have negative exponents, hence it can be a fraction itself rather than a polynomial. Use expandFraction from Product to expand it properly.
