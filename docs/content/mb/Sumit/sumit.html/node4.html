How do I get and install ?

You can download by anonymous ftp from the CAFÉ server at

ftp-sop.inria.fr in cafe/software/sumit
or from the URL:

http://www.inria.fr/cafe/Manuel.Bronstein/sumit/
After downloading the file sumit.tar.gz, issue ``tar -xvzf sumit.tar.gz'' in order to unpack it. This will create the following directories:
  • sumit/doc: this user guide,
  • sumit/lib: the library,
  • sumit/include: the required include files,
  • sumit/test: some test files,
  • sumit/samples: programming samples,
Once the above file is unpacked, do the following:
  • add the option -csys=XXX to your ALDORARGS environment variable, where XXX depends on your hardware and operating system. Common values for XXX are axposf1v4 for OSF1 V4.0 on a DEC Alpha, linux-486 for linux on a 486 or above PC, and sun4os55g-v8 for SunOS 5.5 on a SPARC v8 machine. See the file $ALDORROOT/include/axiomxl.conf for other values;
  • go to the sumit/lib directory and execute source makesumit-csh or source makesumit-bash depending on your shell;
  • if you want to build the GMP version of the library, execute source makesumit.gmp-csh or source makesumit.gmp-bash depending on your shell. See the subsection on using GMP for more information about using the GMP version of ;
  • if you want to build the debug version of the library, execute source makesumitd-csh or source makesumitd-bash depending on your shell; See the subsection on debugging for more information on using the debug library.
