LinearOrdinaryRecurrenceRationalSolutions


Usage

import from LinearOrdinaryRecurrenceRationalSolutions(R, F, , RX, RXE, FX)


Parameter Type Description
R GcdDomain The coefficient ring
  RationalRootRing  
F Field A field containing R
Injection from R into F
RX UnivariatePolynomialCategory R Polynomials over R
RXE LinearOrdinaryRecurrenceCategory RX Recurrences over RX
FX UnivariatePolynomialCategory F Polynomials over F


Description

LinearOrdinaryRecurrenceRationalSolutions(R, F, , RX, RXE, FX) provides a rational solver for linear ordinary recurrence operators with polynomial coefficients.


Exports

kernel: RXE (RX, V UFP) Rational solutions
particularSolution: (RXE, Rx) (RX, Partial UFP) A rational solution
solve: (RXE, Rx) (RX, V UFP, Partial UFP) Rational solutions


where

Rx == Fraction RX
UFP == UnivariateFactorialPolynomial(F, FX)
V == Vector
