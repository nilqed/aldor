How do I use in my programs?

Once is properly built, you need to set the following environment variables before using it:
  • the environment variables ALGEBRAROOT and SUMITROOT should be set respectively to the main algebra and sumit directories;
  • $ALGEBRAROOT/include and $SUMITROOT/include should be appended to your INCPATH environment variable;
  • $ALGEBRAROOT/lib and $SUMITROOT/lib should be appended to your LIBPATH environment variable;
In your ALDOR programs, use #include "sumit" instead of #include "aldor" or #include "algebra". When building your final executable, add the options
-lsumit -lalgebra -laldor -y$SUMITROOT/lib -y$ALGEBRAROOT/lib
to your compiler command line, or
-lsumitd -lalgebrad -laldord -dDEBUG -y$SUMITROOT/lib -y$ALGEBRAROOT/lib
to link to the debug version of . Check the subsection on using GMP for the options required if you want to use the GMP library and the GMP version of . As with any ALDOR program, do not forget the -q option in order to optimize your programs, specially if performance is an issue.

Before using for the first time, please check your installation by running make in the sumit/test directory, followed by running testall.

Please report any installation problem or bugs you encounter to sumit@sophia.inria.fr.
