LinearOrdinaryRecurrenceEigenring


Usage

import from LinearOrdinaryRecurrenceEigenring(R, F, , RX, RXE, FX, FXE)


Parameter Type Description
R GcdDomain The coefficient ring
  RationalRootRing  
F Field A field containing R
Injection from R into F
RX UnivariatePolynomialCategory R Polynomials over R
RXE LinearOrdinaryRecurrenceCategory RX Recurrence operators over RX
FX UnivariatePolynomialCategory F Polynomials over F
FXE LinearOrdinaryRecurrenceCategory FX Recurrence operators over FX


Description

LinearOrdinaryRecurrenceEigenring(R, F, , RX, RXE, FX, FXE) provides eigenring computations for linear ordinary recurrence operators with polynomial coefficients.


Exports

eigenring: RXE (RX, Vector DenseMatrix UFP(F, FX), Vector FXE) Eigenring


where

UFP == UnivariateFactorialPolynomial
