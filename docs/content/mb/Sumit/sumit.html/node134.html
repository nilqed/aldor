LinearOrdinaryDifferentialOperatorSeriesSolutions


Usage

import from LinearOrdinaryDifferentialOperatorSeriesSolutions(R, F, , RX, RXD, Fx)


Parameter Type Description
R IntegralDomain An integral domain
F Field A field containing R
Injection
RX UnivariatePolynomialCategory R Polynomials over R
RXD LinearOrdinaryDifferentialOperatorCategory RX Differential operators
Fx UnivariateTaylorSeriesCategory F A series type over F


Description

LinearOrdinaryDifferentialOperatorSeriesSolutions(R, F, , RX, RXD, Fx) provides a series solver for linear ordinary differential operators with polynomial coefficients around an ordinary point.


Exports

fundamentalMatrix: (RXD, R) DenseMatrix Fx fundamental solution matrix
kernel: (RXD, R) Vector Fx all the series solutions
solution: (RXD, R) Vector F Fx Fx solve an initial value problem
