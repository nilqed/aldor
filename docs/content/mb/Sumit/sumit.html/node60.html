LinearHolonomicDifferentialEquation


Usage

import from LinearHolonomicDifferentialEquation(R, RX, RXD)


Parameter Type Description
R GcdDomain Coefficient domain
  RationalRootRing  
RX UnivariatePolynomialCategory R Polynomials over R
RXD LinearOrdinaryDifferentialOperatorCategory RX Differential operators


Description

LinearHolonomicDifferentialEquation(R, RX, RXD) provides a type that analyzes the local singularities of operators in RXD and stores the local information for eventual reuse.


Exports

allRationalExponents?: % Boolean check local exponents
allRegular?: % Boolean check singularities
analyze!: (%, RX) List SING singularity analysis
analyze!: (%, Product RX) List SING singularity analysis
analyzedSingularities: % List SING analyzed singularities
coerce: RXD % Conversion
finiteSingularities!: % List SING affine singularities
operator: % RXD Underlying operator
order: % Integer Order
singularAtInfinity?: % Boolean check singularity at
singularityAtInfinity!: % SING singularity at


where

SING == PlaneSingularity(R, RX, RXD)
