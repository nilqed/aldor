LinearOrdinaryDifferentialOperatorRationalSolutions


Usage

import from LinearOrdinaryDifferentialOperatorRationalSolutions(R, F, , RX, RXD, FX)


Parameter Type Description
R GcdDomain The coefficient ring
  RationalRootRing  
F Field A field containing R
  RationalRootRing  
Injection
RX UnivariatePolynomialCategory R Polynomials over R
RXD LinearOrdinaryDifferentialOperatorCategory RX Differential operators
FX UnivariatePolynomialCategory F Polynomials over F


Description

LinearOrdinaryDifferentialOperatorRationalSolutions(R, F, , RX, RXD, FX) provides a rational solver for linear ordinary differential operators with polynomial coefficients.


Exports

kernel: EQ (Product RX, V FX) Rational solutions
  (EQ, RX) (Product RX, V FX)  
  RXD (Product RX, V FX)  
  (RXD, RX) (Product RX, V FX)

solve: (RXD, Rx) (Product RX, V FX, P FX) Rational solutions


where

EQ == LinearHolonomicDifferentialEquation(R, RX, RXD)
Rx == Fraction RX
V == Vector
P == Partial
