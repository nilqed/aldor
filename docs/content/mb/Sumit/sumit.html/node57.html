FractionFreeSymmetricPower


Usage

import from FractionFreeSymmetricPower(R, Rd, n, m, P, irr?)


Parameter Type Description
R IntegralDomain The coefficient domain
Rd LinearOrdinaryDifferentialOperatorCategory R A LODO type over R
n MachineInteger The power to compute
m MachineInteger Order of the operator
P R Leading coefficient
irr? Boolean true for irreducible P


Description

FractionFreeSymmetricPower(R, Rd, n, m, P, irr?) provides functions that compute exterior and symmetric powers of operators of order m having P as leading coefficient. This package should not be used when P is a unit in R, in which case SymmetricPower should be used instead (for operators of order at least 3). It should also not be used for , for which a faster algorithm is provided by SecondOrderSymmetricPower.


Exports

exteriorPower: Rd Rd exterior power
exteriorPowerSystem: Rd DenseMatrix R exterior power
symmetricPower: Rd Rd symmetric power
symmetricPowerSystem: Rd DenseMatrix R symmetric power
