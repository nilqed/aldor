banner


Usage

banner(p, s)


Signature

banner: (TextWriter, String) TextWriter


Parameter Type Description
p TextWriter The port to write to
s String A name to center in the banner


Description

Writes the default  banner to the port p.


Remarks

The string s is automatically centered in the banner. In order for the banner to look ok, it must have a maximum of 33 characters.


Example

banner(stdout, "B E R N I N A");
writes the banner 
                                                          /\
      -----  it                                          /| \
      \                   B E R N I N A                 / |  \
       \                                               |   \  \
       /                 A Sum^it server               |    |  \
      /                                                /     \  \
      -----                                           /      |   \
                                                     /       /    \
to stdout.
