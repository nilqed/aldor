SymmetricPower


Usage

import from SymmetricPower(R, Rd, n, m)


Parameter Type Description
R IntegralDomain The coefficient domain
Rd LinearOrdinaryDifferentialOperatorCategory R A LODO type over R
n MachineInteger The power to compute
m MachineInteger Order of the operator


Description

SymmetricPower(R, Rd, n, m) provide functions that compute exterior and symmetric powers of operators of order m having invertible leading coefficients. This package should not be used for , for which a faster algorithm is provided by SecondOrderSymmetricPower. Use FractionFreeSymmetricPower for operators of order at least whose leading coefficients are not invertible in .


Exports

exteriorPower: (Rd, R) Rd exterior power
exteriorPowerSystem: (Rd, R) DenseMatrix R exterior power
symmetricPower: (Rd, R) Rd symmetric power
symmetricPowerSystem: (Rd, R) DenseMatrix R symmetric power
