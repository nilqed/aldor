Profiling and debugging

The macros TIME, TRACE and AGAT provided by libaldor remain available in . In addition, also comes with a debug version, which makes many assertions about the arguments of the functions called as well as their results. While those assertions slow down the code considerably they tend to be quite useful when chasing bugs since the release version of does not make validity checks to most of its inputs. The debug version of must be used jointly with the debug versions of libaldor and Algebra. To use them, just compile your application with the
-lsumitd -lalgebrad -laldord -dDEBUG
options rather than -lsumit -lalgebra -laldor. It is also preferable when debugging to add the -q1 option in order to prevent inlining.
