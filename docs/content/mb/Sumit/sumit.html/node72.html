exteriorPower,symmetricPower


Usage

exteriorPower(L, n)
symmetricPower(L, n)


Signature

exteriorPower,symmetricPower: (%, Integer) %


Parameter Type Description
L % A differential operator
n Integer A positive integer


Description

exteriorPower() returns a differential operator of minimal order whose kernel is generated by the Wronskians of elements of a basis of , while symmetricPower() returns a differential operator of minimal order whose kernel is generated by the all the products of elements of a basis of .
