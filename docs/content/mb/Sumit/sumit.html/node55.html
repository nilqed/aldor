FractionFreeCyclicVector


Usage

import from FractionFreeCyclicVector(R, P, irr?)


Parameter Type Description
R IntegralDomain The coefficient domain
P R a nonzero nonunit of R
irr? Boolean Indicates whether P is known to be irreducible in R


Description

FractionFreeCyclicVector(R, P, irr?) provides linear algebra tools for fraction-free computations of cyclic vectors and symmetric and exterior powers.


Exports

firstDependence: DenseMatrix FractionBy(R, P, irr?) Vector R Dependence
