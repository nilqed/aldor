Remarks

particularSolution(L, g) and particularSolution(L, []) are available only when the ring R has RationalRootRing. Produces an error if is neither a differential or recurrence operator. The polynomials and are expressed in terms of the basis described in recurrence. Using the version with a vector of right-hand sides is more efficient than calling repeatedly the version with a single right-hand side.
