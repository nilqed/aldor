newtonPolygon


Usage

newtonPolygon(f,p)


Signature

newtonPolygon: (Rxy, Rx) %


Parameter Type Description
f Rxy a bivariate polynomial over R
p Rx the place where to localize (optional)


Returns

Returns the algebraic Newton polygon of f at any root of p, at infinity if p is omitted.


Remarks

This function requires that the Newton polygons at all the roots of p be the same. This is always the case when p is irreducible over R, or balanced with respect to all the coefficients of f. This assumption is not checked by this function, the resulting polygon will be wrong if it is not satisfied.
