LinearOrdinaryOperatorToRecurrence


Usage

import from LinearOrdinaryOperatorToRecurrence(R, Rx, Rxy, Rxe);


Parameter Type Description
R CommutativeRing A commutative ring
Rx UnivariatePolynomialCategory R Polynomials over R
Rxy UnivariateSkewPolynomialCategory Rx Operators over Rx
Rxe LinearOrdinaryRecurrenceCategory Rx Recurrence operators over Rx


Description

LinearOrdinaryOperatorToRecurrence(R, Rx, Rxy, Rxe) provides conversions from linear operators to recurrences.


Exports

drop: Rxy Integer degree drop
recurrence: Rxy (Rxe, Integer) recurrence for some basis
  SYS Rx (Array DenseMatrix Rx, Integer)  


where

SYS == LinearOrdinaryFirstOrderSystem
