kernel


Usage

kernel L
kernel(L, n)


Signatures

kernel: SYS RX DenseMatrix FX
kernel: (SYS RX, Integer) DenseMatrix FX


where

SYS == LinearOrdinaryFirstOrderSystem


Parameter Type Description
L SYS RX A linear ordinary first order differential or difference system
n Integer A degree bound (optional)


Description

kernel(L) returns a matrix whose columns for a basis for all the polynomial solutions of , while kernel(L,n) returns a matrix whose columns form a basis for the polynomial solutions of of degree at most .


Remarks

kernel(L) is available only if R has RationalRootRing. The type of system (differential or difference) is determined from the parameter RXY to the domain. Produces an error if RXY is neither a differential or recurrence operator type. The polynomials in the kernel are expressed in terms of the basis described in recurrence.
